# Process raw data for use in pMCMC

`data_process` Transform data file

## Usage

``` r
data_process(data_raw = NULL, start_pf_time = NULL, check_flexibility)
```

## Arguments

- data_raw:

  Data input from user Default = NULL

- start_pf_time:

  Amount of time before first observation to start particle filter

- check_flexibility:

  Save pre-observation run
